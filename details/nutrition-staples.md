# Nutrition Reference — Warren's Staples

## Smoothie Ingredients (always same brands)
- **Silk Unsweetened Soy Milk:** 40 cal/100ml — 3g protein, 3.2g carbs, 1.6g fat, 0.7g fibre, 2g sugar per 100ml
- **Naked Whey Protein Concentrate (unflavoured):** 120 cal/30g — 25g protein, 3g carbs, 2g fat, 0g fibre, 2g sugar, 135mg sodium per label *(upgraded from Manitoba Harvest Hemp Yeah! as of 2026-05-18)*
- **Manitoba Harvest Hemp Yeah! Protein Powder (old):** 55 cal/15g — 7.5g protein, 4g carbs, 4g fibre, 0g sugar, 1.5g fat
- **Lifestyle Psyllium Husk Fibre:** 33 cal/10g — 0g protein, 8g carbs, 8g fibre, 0g fat

## Standard Smoothie Recipe (Canada)
24g banana (half) + 98g strawberries + 118g blueberries + 300ml Silk soy milk + 10g psyllium + 30g Naked Whey
→ **~392 cal / 36g protein / 51g carbs / 7g fat / 16g fibre / 29g sugar / ~175mg sodium**

## Dressings
- **Farm Boy Apple Cider Garlic Vinaigrette:** 300 cal/100ml — 0.7g protein, 3.3g carbs, 30g fat, 0g fibre, 0g sugar, ~117mg sodium per 100ml (per 30ml serving: 90 cal, 9g fat, 1g carbs, 0g sugar, 35mg sodium)

## Fish
- **Farm Boy Pacific Cod Fillet (105g/fillet):** 220 cal / 13g protein / 15g carbs / 12g fat / 1g fibre / 0g sugar / 390mg sodium

## Salads
- **Farm Boy Superfood Blend (450g bag):** 100 cal / 4g protein / 10g carbs / 5g fat / 3g fibre / 4g sugar / 55mg sodium per 105g serving

## Other Staples
- **Grillies "Grilled Killer" chicken burrito (521g):** 825 cal / 67g protein / 82g carbs / 22g fat / 10g fibre / 8g sugar — whole wheat tortilla, grilled chicken, mozzarella, mild beans, veggies, salsa. No rice, no sour cream.
- **Grillies bowl (salad form, no tortilla, ~400g):** ~625 cal / 60g protein / 47g carbs / 19g fat / ~6g fibre — same ingredients as burrito minus tortilla. No avocado by default — costs extra, worth adding (+~150 cal / +5g fibre). "Secret sauce" = burrito sauce (chipotle/mayo base).
- **Grillies bowl DOUBLE chicken (~550g):** ~900 cal / 90g protein / 47g carbs / 25g fat / ~6g fibre — same as above with extra chicken portion. Warren's standard order.
- **Country's Finest hot sausage sticks:** 65 cal / 4g protein / 5g fat per stick (15g)
- **Noriko's minced turkey dish:** ~98 cal/100g, ~13g protein/100g
- **Warren's turkey/pork chili (always low-sodium, always double meat):** ~108 cal/100g, 10.3g protein, 8.2g carbs, 4.0g fat, ~179mg sodium per 100g. Yield: ~2,580g per batch.
