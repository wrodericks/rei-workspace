# Use Case: Domain Expert Agent (IFRS 17 / Actuarial)

- Build an agent with deep knowledge of a specific actuarial domain
- Primary focus: IFRS 17
- Potential commercial product — exploring feasibility first
- Key insight: value lives in encoding expertise (the gaps, debates, common mistakes) not just the standard text
- Starting point: narrow experiment on a slice of IFRS 17 (e.g. GMM or contract boundaries)
